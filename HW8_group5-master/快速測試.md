## 🎯 立即測試（3分鐘）

### 1️⃣ 打開網站
在瀏覽器訪問：**http://localhost:8080**

### 2️⃣ 測試基礎搜尋
在搜尋框輸入以下關鍵字，按Enter：
- `不限時` ➜ 應該顯示不限時的咖啡廳
- `安靜` ➜ 應該顯示安靜的咖啡廳  
- `有插座` ➜ 應該顯示有插座的咖啡廳
- `wifi` ➜ 應該顯示有wifi的咖啡廳

### 3️⃣ 測試篩選功能
1. 點擊「地區篩選」按鈕

1. 在任一搜尋結果點擊 ❤️ 按鈕
2. 點擊右上角「收藏清單」
3. 確認咖啡廳已加入收藏

### 5️⃣ 測試熱門推薦

## 🔬 API 測試（開發者用）

在 PowerShell 執行以下指令：

### 測試搜尋 API
```powershell
# 搜尋「不限時」
Invoke-RestMethod -Uri "http://localhost:8080/api/search?q=不限時" -Method GET

# 搜尋「安靜」
Invoke-RestMethod -Uri "http://localhost:8080/api/search?q=安靜" -Method GET

# 進階搜尋（大安區 + 不限時）
Invoke-RestMethod -Uri "http://localhost:8080/api/search/advanced?keyword=咖啡&districts=大安區&features=不限時" -Method GET
```

### 測試推薦 API
```powershell
# 取得熱門推薦
Invoke-RestMethod -Uri "http://localhost:8080/api/recommendations" -Method GET

# 取得大安區推薦
Invoke-RestMethod -Uri "http://localhost:8080/api/recommendations?districts=大安區" -Method GET
```

### 測試篩選器 API
```powershell
# 取得所有地區
Invoke-RestMethod -Uri "http://localhost:8080/api/filters/districts" -Method GET

# 取得所有功能
Invoke-RestMethod -Uri "http://localhost:8080/api/filters/features" -Method GET
```

## 🐛 常見問題

### Q1: 搜尋沒有結果？
**解決方式：**
- 檢查是否輸入了停用詞（的、是、在等）
- 嘗試使用更常見的關鍵字（不限時、安靜、wifi）
- 查看 Chrome DevTools Console 的錯誤訊息

### Q2: 分數都一樣？
**檢查：**
- 確認 `keywords.json` 的權重設定
- 確認 `cafes.json` 包含正確的關鍵字
- 嘗試使用不同的關鍵字組合

### Q3: 篩選沒有作用？
**檢查：**
- 確認已經點擊「套用」按鈕
- 確認選擇的地區/功能有對應的咖啡廳
- 嘗試清除篩選再重新選擇
